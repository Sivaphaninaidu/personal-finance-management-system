# Personal Finance Management System

A full-stack web application for managing personal finances, including accounts, transactions, expenses, bills, budgets, goals, analytics, and reports.

## Features

* User Registration & Login
* Account Management
* Transaction Tracking
* Expense Management
* Bill Management
* Goal Tracking
* Budget Management
* Dashboard Analytics
* PDF Report Generation
* Dark Mode Support
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* Recharts
* CSS

### Backend

* FastAPI
* Django ORM
* SQLite (Current)
* PostgreSQL (Planned)

## Installation

### Clone Repository

git clone https://github.com/Sivaphaninaidu/personal-finance-management-system.git

### Frontend

cd finance

npm install

npm run dev

### Backend

cd fastapi_backend

pip install -r requirements.txt

uvicorn main:app --reload
